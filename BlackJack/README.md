# 🂡 Blackjack 🂫

You will play a card game, **Blackjack**, against a computer‑controlled dealer.

This program was written as an example for entry‑level Python/Streamlit programmers, so the game itself is simplified to the bare basics. It judges only by the total value of the cards. There are no special rules or side bets. It uses only a single deck. The dealer’s logic is also simple and conservative: he draws cards only when his hand is below 17.

The code is written in vanilla Python without any external dependencies except streamlit (`requirment.txt` is not required as it is pre-installed in Streamlit Community Cloud). UTF‑8 characters are used for the card images, so there are no image‑file dependencies either (see [“Playing cards in Unicode”](https://en.wikipedia.org/wiki/Playing_cards_in_Unicode) on Wikipedia).
